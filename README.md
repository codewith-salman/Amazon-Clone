## 📖 About The Project

<img src="Amazon%20Clone/images/screenshot.PNG" alt="Amazon Clone Screenshot" width="100%" />

The **Amazon Clone** is a front-end project inspired by Amazon’s homepage design.  
The main goal of this project was to practice **layout structuring**, **product grid systems**, and **UI styling** using pure HTML and CSS.

This project focuses on:
- Header and navigation layout  
- Product grid and card structure  
- Clean spacing and alignment  
- Responsive-friendly design approach  

Ideal for:
- Front-end practice projects  
- UI cloning exercises  
- Portfolio showcasing  
- HTML & CSS skill improvement  

---

## ✨ Features

- Amazon-style landing page UI  
- Clean and structured layout  
- Product cards and grid system  
- Hover and focus effects  
- Easy-to-read and organized code
